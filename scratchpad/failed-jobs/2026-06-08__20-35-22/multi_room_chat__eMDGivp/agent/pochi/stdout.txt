### How It Works

- **Room isolation**: Each `roomId` creates a separate Durable Object instance via `useSyncedState`'s `roomId` parameter, which scopes the state to `/__synced-state/${roomId}`. Different rooms have completely independent message histories.

- **Realtime sync**: The `useSyncedState` hook automatically subscribes to state changes, so all viewers of the same room see updates in real time via the capnweb RPC mechanism.
