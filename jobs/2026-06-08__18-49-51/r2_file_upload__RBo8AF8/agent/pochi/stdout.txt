| Endpoint | Behaviour |
|---|---|
| `POST /api/files` | Reads the `file` field from `multipart/form-data`, computes `SHA-256` of the raw bytes, stores the object under `uploads/<sha256hex>.<ext>` (or `uploads/<sha256hex>` if no extension), returns HTTP 201 JSON with `key`, `size`, `contentType`, `sha256`. Returns 400 if `file` field is absent. |
| `GET /api/files` | Lists all objects under the `uploads/` prefix; returns `{ objects: [{ key, size, uploaded }] }` (ISO-8601 timestamps). |
| `GET /api/files/:key` | URL-decodes the key, fetches the object from R2, streams the raw bytes with the original `Content-Type` and `Content-Length`. Returns 404 if missing. |
| `DELETE /api/files/:key` | URL-decodes the key, checks existence with `BUCKET.head()`, deletes the object, returns HTTP 204 empty body. Returns 404 if missing. |
