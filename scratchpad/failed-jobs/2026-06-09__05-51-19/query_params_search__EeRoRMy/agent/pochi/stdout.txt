### `GET /search`
Parses query parameters using the standard `URL` API and returns a JSON response with:
- **`q`**: the value of the `q` parameter, or `""` if missing
- **`tags`**: array of all values for repeated `tag` parameters (via `URLSearchParams.getAll()`)
- **`page`**: integer from `page` param, defaults to `1` if missing or non-numeric
- **`limit`**: integer from `limit` param, clamped to `[1, 100]`, defaults to `10` if missing

### `GET /echo`
Returns a JSON object with every query parameter as a string key. For duplicate keys, only the first value is used (by checking `key in result` before setting).
